# FPGA Projects

This repository contains my FPGA-based designs implemented using Verilog on boards like Arty A7 and ZedBoard.

The purpose of this repository is to document my learning and hands-on experience with digital design and FPGA development.

---

## 🧠 Focus Areas

- Verilog-based digital design
- Understanding FPGA design flow (synthesis, implementation, bitstream generation)
- Basic hardware testing on development boards
- Working with Xilinx Vivado tools
- Building small functional modules and systems

---

## 📁 Projects Overview

### Arty A7

- **LED Blink**  
  Initial project to verify board setup and clock configuration.

- **Up-Down Counter**  
  Counter with increment and decrement control.

- **Up-Down Counter (IP Catalog)**  
  Same functionality implemented using Vivado IP Integrator.

- **Traffic Light Controller**  
  Simple FSM-based design.

- **Vector-Scalar Multiplier**  
  Basic arithmetic operation using Verilog.

- **DIP Switch Multiplication Table**  
  Displays multiplication output based on input switches.

- **Adder-Subtractor**  
  Performs addition and subtraction based on control signal.

---

### ZedBoard

- **LED Blink**  
  Basic test project for Zynq-based board.

---
